Dataset Requirements

  • Data are in the smart cities domain

    • In this link can be seen the following text:

    Los datos abiertos forman una estrategia de reutilización de datos por parte de las administraciones públicas.

    Which confirms that this first requirements is satisfied.

  • Data have an open license so it can be published

    • As can be seen in the following link, the data can be used by any person or company.

    Canarias Open Data

  • Data are available as a CSV file

    • Most of the open data published in the web are available as CSV files. We have chosen three CSV files (monumentoshistoricos.csv, monumentosnaturales.csv, oficinasdeturismo.csv)

    Datasets published

  • Data can be easily linked with generic real-world entities

    • The CSV files contain all kind of data that can be identified as entities of the real world, for instance, we can see locations (localización), web pages (web), name (nombre), and so on.

  • Documentation exists for the data (optional)
  • Data can come from multiple data sources (optional)

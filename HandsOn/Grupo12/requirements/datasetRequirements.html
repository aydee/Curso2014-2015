Requirements of the dataset

Data are in the smart cities domain

Madrid is one of the smart cities given by the teacher. This dataset is uploaded by the Ayuntamiento de Madrid which is the City Hall of Madrid.

Data are available as a CSV file

The dataset is in the open data portal Portal de datos abiertos. The format given depends on each dataset, in this case the dataset is given as:

  • CSV file
  • GEO file
  • RDF file
  • XML file

Data have an open license so it can be published

The license of the data is open and we can found all the license in Condiciones de uso

Data can be easily linked with generic real-world entities (e.g., locations)

The data can be linked with dddress data, schedule, and coordinate services for georeferencing.

Documentation exists for the data (optional).

There is documentation about the diferent formats given, and about the metadatas. Also in the XML file we can found a description of each element given and for each library and bookmobile

Data can come from multiple data sources (optional).

All the data given comes from the web www.madrid.es, so that condition is not fulfilled
